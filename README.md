# Anti-ROOTKIT


 Полезный мульти инструмент AntiRootkit для сканирование процесов -->
  Для обнаружение каких либо хуков, драйверов, процесов
  помогающее лучше анализировать процесов и просматривать/упровлять функциями!



#Function
  удобен в плане сканировании PE файлов, проверка и обнаружение IAT - хуков, сканирование драйверов 
  также полное сканирование всего, в конце покажет полнотстью все выявление!

#technology
вообще мой проект написан на ассемблере, почти все файлы это асм и почти за всю логику отвечает асм
за исключением main.cpp - он в проекте как связующая часть всех асм файлов
