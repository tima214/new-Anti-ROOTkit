# AntiRootkit with Emulator

Антируткит для обнаружения скрытых процессов, драйверов и признаков работы Rootkit в операционной системе Windows с использованием безопасной эмуляции подозрительного кода.

---

## GitHub

Репозиторий проекта:

https://github.com/tima214/new-Anti-ROOTkit

---

## Технологии

- C++
- MASM x64 Assembly
- Windows API
- ToolHelp32 API
- PSAPI
- PE (Portable Executable) Parser
- Win32 API
- Visual Studio 2022

---

# Быстрый старт

## 1. Клонирование проекта

```bash
git clone https://github.com/tima214/new-Anti-ROOTkit.git
cd new-Anti-ROOTkit
```

---

## 2. Требования

- Windows 10 / Windows 11
- Visual Studio 2022
- Desktop development with C++
- MASM Build Tools
- Windows SDK

---

## 3. Сборка проекта

Откройте проект в **Visual Studio 2022**.

Выберите конфигурацию:

```
Release x64
```

или

```
Debug x64
```

Нажмите:

```
Build → Build Solution
```

или

```
Ctrl + Shift + B
```

---

## 4. Запуск

После успешной сборки запустите программу:

```
AntiRootkit.exe
```

или используйте:

```
Ctrl + F5
```

# Модули

## main.cpp

Основной модуль приложения.

Отвечает за:

- запуск программы;
- вызов модулей проверки;
- обработку результатов;
- вывод информации пользователю.

---

## process_scan.asm

Сканирование процессов.

Возможности:

- получение списка процессов;
- поиск скрытых процессов;
- анализ PID;
- сравнение процессов.

---

## driver_scan.asm

Проверка драйверов.

Функции:

- получение списка драйверов;
- поиск неизвестных драйверов;
- проверка загруженных модулей;
- обнаружение скрытых драйверов.

---

## memory_scan.asm

Анализ памяти.

Выполняет:

- сканирование виртуальной памяти;
- поиск подозрительных участков;
- анализ защиты страниц;
- обнаружение внедренного кода.

---

## pe_scan.asm

Проверка Portable Executable.

Проверяются:

- DOS Header
- NT Header
- Sections
- Entry Point
- Import Table
- Export Table

---

## iat_check.asm

Проверка Import Address Table.

Определяет:

- подмену адресов функций;
- API Hooking;
- изменение таблицы импорта;
- подозрительные модули.
