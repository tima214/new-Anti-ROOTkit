#Антируткит для Windows

В рамках проекта был разработан антируткит, предназначенный для обнаружения признаков скрытого вредоносного программного обеспечения в операционной системе Windows.
Программа использует возможности WinAPI для анализа состояния системы и выявления потенциально опасных изменений.

Антируткит выполняет несколько видов проверок, включая анализ загруженных драйверов, поиск подозрительных записей в таблице импорта (IAT),
а также другие проверки, направленные на обнаружение признаков присутствия руткитов. Полученные результаты выводятся в удобном текстовом виде, что позволяет быстро оценить состояние системы и выявить возможные угрозы.

При разработке проекта были использованы язык программирования C++, WinAPI и структуры формата PE,
что позволило изучить принципы работы операционной системы Windows, механизмы загрузки модулей и методы обнаружения некоторых техник сокрытия вредоносного кода.
